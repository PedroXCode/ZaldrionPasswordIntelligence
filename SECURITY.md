# Security Policy

Zaldrion Password Intelligence is a local Windows desktop tool. It does not send generated passwords to a server and it does not save the session history automatically.

## Important notes

- Exporting the session history creates a plain-text `.txt` file. Use that feature only for testing or when you can store the file securely.
- The cracking time is an educational estimate, not a guarantee.
- Do not reuse the same password across multiple accounts.
- For real accounts, use a password manager and multi-factor authentication.

## Reporting issues

Open a GitHub issue with clear reproduction steps. Do not post real passwords, tokens, private keys, or personal data.
