# Changelog

## 1.0.0 - 2026-05-01

### Added
- Modern WPF interface with dark cyber style.
- Cryptographic password generation using `RandomNumberGenerator`.
- Password analysis with score, entropy, detected character groups and estimated cracking time.
- Weak-pattern checks: short length, repeated characters, repeated patterns, common terms and keyboard sequences.
- Session-only password history.
- Optional export to `.txt` with warning.
- GitHub Actions workflow for Windows build artifact.
